# CRS 资产信息交换风险评估 — 问卷与必填字段定义

本文件定义五类资产的分级问题结构、每个问题的必填字段清单、触发条件和跳转逻辑。SKILL.md 在执行完整性校验时以本文件为唯一依据。

**通用处理规则：**
- 对于任何必填字段，若用户回答"不清楚"，记录为"信息缺失：用户不确定"，对应资产标注"存在风险，[字段名]不明确"，继续推进后续问题。
- 对于任何必填字段，若用户拒绝回答，记录为"用户拒绝提供"，对应资产标注"存在风险，相关信息不完整"，继续推进后续问题。
- 每个字段均作为独立问题单独提出，不得打包。

---

## 一、银行存款 / 金融账户

### 一级问题

**问题内容**：您是否持有海外银行存款或金融账户（含定期存款、活期账户、股票账户、理财账户、基金账户等）？

**字段**：是否持有，必填，选项：是 / 否

**跳转规则**：回答否 → 跳至第二类资产（上市公司股票）

---

### 二级问题（回答是后，逐题依次询问）

**字段 1**：账户所在国家或地区
- 类型：自由输入
- 必填：是
- 用途：查询 crs-member-countries.md 中的激活状态
- 提问参考："请问该账户开设在哪个国家或地区？"

**字段 2**：账户类型
- 类型：选项
- 选项：个人账户 / 联名账户 / 公司账户 / 信托账户
- 必填：是
- 触发规则：选择"公司账户"或"信托账户"时，触发三级问题
- 提问参考："请问该账户是个人账户、联名账户、公司账户，还是信托账户？"

**字段 3**：账户余额大致范围
- 类型：选项
- 选项：50 万人民币以下 / 50 万至 300 万人民币 / 300 万人民币以上
- 必填：是
- 用途：用于报告展示
- 提问参考："请问该账户的余额大致在哪个范围？50 万以下、50 万到 300 万，还是 300 万以上？"

---

### 三级问题（仅当账户类型为"公司账户"或"信托账户"时触发，逐题依次询问）

**字段 4**：账户所属机构的主营业务类型
- 类型：选项
- 选项：投资理财（如基金管理、资产配置等）/ 实际经营业务（如贸易、生产、服务等）/ 不清楚
- 必填：是
- 提问参考："请问持有该账户的公司或信托，其主要业务是投资理财，还是实际经营业务？"

**字段 5**：被动收入占比是否超过 50%
- 类型：选项
- 选项：是，超过 50% / 否，低于 50% / 不清楚
- 必填：是
- 补充说明：被动收入包括股息、利息、租金、资本利得等
- 提问参考："该公司或信托上一年度的收入中，股息、利息、租金、资本利得等被动收入的比例是否超过 50%？"

---

### 多账户处理

所有字段收集完毕后询问："请问您还有其他海外银行存款或金融账户需要一并评估吗？"若有，重新从二级问题开始收集，独立处理。

## 二、上市公司股票

### 一级问题

**问题内容**：您是否持有海外上市公司的股票？

**字段**：是否持有，必填，选项：是 / 否

**跳转规则**：回答否 → 跳至第三类资产（非上市公司股权）

---

### 二级问题（回答是后，逐题依次询问）

**字段 1**：股票所在交易所
- 类型：自由输入
- 必填：是
- 提问参考："请问该股票在哪个交易所上市？（例如香港联交所、纽约证券交易所等）"

**字段 2**：股票所在国家或地区
- 类型：自由输入
- 必填：是
- 用途：查询 crs-member-countries.md
- 提问参考："该交易所所在的国家或地区是哪里？"

**字段 3**：持有方式
- 类型：选项
- 选项：个人直接持有 / 通过券商账户持有 / 通过公司或信托持有
- 必填：是
- 触发规则：选择"通过公司或信托持有"时，触发三级问题
- 提问参考："请问您是个人直接持有这些股票，通过券商账户持有，还是通过公司或信托持有？"

---

### 三级问题（仅当持有方式为"通过公司或信托持有"时触发，逐题依次询问）

**字段 4**：持有架构的主营业务类型
- 类型：选项
- 选项：投资理财（主要持有金融资产）/ 实际经营业务 / 不清楚
- 必填：是
- 提问参考："持有这些股票的公司或信托，其主要业务是投资理财还是实际经营？"

**字段 5**：被动收入占比是否超过 50%
- 类型：选项
- 选项：是，超过 50% / 否，低于 50% / 不清楚
- 必填：是
- 提问参考："该公司或信托上一年度，被动收入（如股息、利息、资本利得）是否超过总收入的 50%？"

- 类型：自由输入

**字段 2**：持有方式

**字段 6**：是否有员工和实质经营活动
- 类型：选项
- 选项：有员工和实际经营活动 / 基本没有，主要是持股或持有资产 / 不清楚
- 必填：是
- 提问参考："该公司是否有实际的员工、办公场所和日常经营活动？"

## 四、保险（有现金价值的保单）

### 一级问题

**问题内容**：您是否持有海外保险保单？

**字段**：是否持有，必填，选项：是 / 否

**跳转规则**：回答否 → 跳至第五类资产（信托）

---

### 一级补充问题（回答是后，立即触发此筛选问题）

**问题内容**：该保单是否具有现金价值？具有现金价值的保险包括终身寿险、年金险、储蓄型保险等；纯消费型寿险（如定期寿险）、医疗险、意外险一般不具有现金价值。

**字段**：是否具有现金价值，必填，选项：是 / 否 / 不确定

**跳转规则**：
- 回答否 → 说明"该保单不具有现金价值，不在 CRS 信息交换范围内"，跳至第五类资产（信托），不再询问后续字段。
- 回答不确定 → 记录为"现金价值状态不明确"，在报告中标注"存在风险，建议确认保单是否具有现金价值"，跳至第五类资产。
- 回答是 → 继续进入二级问题。

**提问参考**："请问该保单是否具有现金价值？比如终身寿险、年金险、储蓄型保险一般有现金价值；纯消费型的定期寿险、医疗险一般没有。"

---

### 二级问题（通过现金价值筛选后，逐题依次询问）

**字段 1**：保险公司所在国家或地区
- 类型：自由输入
- 必填：是
- 用途：查询 crs-member-countries.md
- 提问参考："请问该保险公司注册在哪个国家或地区？"

**字段 2**：保单类型
- 类型：选项
- 选项：终身寿险 / 年金险 / 储蓄型保险 / 其他
- 必填：是
- 提问参考："请问这是哪种类型的保单？终身寿险、年金险、储蓄型保险，还是其他类型？"

**字段 3**：投保人身份
- 类型：自由输入
- 必填：是
- 提问参考："请问该保单的投保人（保单持有人）是谁？是您本人，还是其他个人、公司或信托？"

**字段 4**：被保险人身份
- 类型：自由输入
- 必填：是
- 提问参考："请问该保单的被保险人是谁？"

**字段 5**：受益人身份
- 类型：自由输入
- 必填：是
- 提问参考："请问该保单的受益人是谁？"

**字段 6**：是否有公司或信托参与
- 类型：选项
- 选项：是（投保人、被保险人或受益人中有公司或信托）/ 否
- 必填：是
- 触发规则：选择是时，触发三级问题
- 提问参考："投保人、被保险人或受益人这几个角色中，是否有通过公司或信托来持有或安排的？"

---

### 三级问题（仅当有公司或信托参与时触发，逐题依次询问）

**字段 7**：涉及架构的注册地 / 设立地
- 类型：自由输入
- 必填：是
- 提问参考："涉及的公司或信托，注册地或设立地在哪个国家或地区？"

**字段 8**：涉及架构的主营业务类型
- 类型：选项
- 选项：投资理财 / 实际经营 / 不清楚
- 必填：是
- 提问参考："该公司或信托，主要从事的是投资理财还是实际经营？"

**字段 9**：涉及架构的被动收入占比
- 类型：选项
- 选项：是，超过 50% / 否，低于 50% / 不清楚
- 必填：是
- 提问参考："该公司或信托，被动收入是否超过其总收入的 50%？"

## 五、信托

### 一级问题

**问题内容**：您是否参与设立了海外信托，或者作为受益人、受托人、保护人、监察人参与其中？

**字段**：是否参与，必填，选项：是 / 否

**跳转规则**：回答否 → 五类资产收集完毕，进入报告生成阶段

---

### 二级问题（回答是后，逐题依次询问）

**字段 1**：信托设立地
- 类型：自由输入
- 必填：是
- 用途：查询 crs-member-countries.md
- 提问参考："请问该信托设立在哪个国家或地区？"

**字段 2**：信托类型
- 类型：选项
- 选项：可撤销信托 / 不可撤销信托 / 全权信托（Discretionary Trust）/ 不清楚
- 必填：是
- 提问参考："请问该信托是可撤销信托、不可撤销信托，还是全权信托（也叫 Discretionary Trust）？"

**字段 3**：用户在信托中的角色（可多选）
- 类型：多选
- 选项：委托人（设立人）/ 受益人 / 受托人 / 保护人 / 监察人 / 其他
- 必填：是
- 提问参考："请问您在该信托中担任什么角色？委托人、受益人、受托人、保护人、监察人，还是其他角色？可以多选。"

---

### 三级问题（始终触发，逐题依次询问）

**字段 4**：信托主要持有的资产类型（可多选）
- 类型：多选
- 选项：银行存款或金融账户 / 上市公司股票 / 非上市公司股权 / 保险 / 其他资产
- 必填：是
- 提问参考："请问该信托主要持有哪些类型的资产？"

**字段 5**：信托是否属于投资机构
- 类型：选项
- 选项：是（主要业务为投资金融资产）/ 否 / 不清楚
- 必填：是
- 提问参考："请问该信托是否属于投资机构，即主要业务是投资和管理金融资产？"

---

### 四级问题（仅当信托持有"非上市公司股权"时触发，逐题依次询问）

**字段 6**：被投资公司的性质
- 类型：选项
- 选项：实际经营公司（有正常经营业务和员工）/ 投资壳公司（主要持有资产，无实质经营）/ 不清楚
- 必填：是
- 提问参考："信托持有的这家非上市公司，是有实际业务和员工的经营公司，还是主要用于持有资产的投资壳公司？"
