## 简介

CRS信息披露诊断Skill是一个智能问答工具，帮助拥有境外资产的中国税收居民评估**CRS（Common Reporting Standard，共同申报准则）信息交换风险**，并生成详细的诊断报告和合规建议。

### 核心功能

- 📋 **智能问答**：通过37个结构化问题，逐步收集境外资产信息
- 🎯 **风险评估**：基于CRS规则和中国税法，判定信息披露义务和风险等级
- 📊 **报告生成**：输出完整的诊断报告，包含风险分析、税务处理和合规建议
- 💡 **专业指导**：提供可操作的合规措施和专业咨询建议

### 适用场景

- ✅ 拥有境外银行存款的中国税收居民
- ✅ 持有境外证券投资账户
- ✅ 设立或控制离岸公司（BVI、开曼、香港等）
- ✅ 购买境外保单（有现金价值）
- ✅ 作为离岸信托的委托人、受益人或保护人
- ✅ 计划进行跨境资产配置前的风险评估
- ✅ 已收到税务机关询问需要应对

### 支持的资产类型

| 资产类型 | 说明 | 问题数量 |
|---------|------|---------|
| 境外银行存款 | 境外银行的储蓄、活期、定期账户等 | 8个 |
| 境外证券投资 | 境外证券账户、股票、基金、债券等 | 5个 |
| 离岸公司 | BVI、开曼、香港等地注册的公司 | 12个 |
| 境外保单 | 有现金价值或年金性质的境外保险 | 7个 |
| 离岸信托 | 境外设立的家族信托、资产信托等 | 12个 |

## 快速开始

### 第一次使用

1. **打开Cursor IDE**
2. **启动Skill**：在Cursor中输入 `@CRS信息披露诊断` 或选择该Skill
3. **开始诊断**：Skill会自动展示欢迎语和第一个问题
4. **按提示回答**：逐个回答问题，Skill会自动引导整个流程
5. **获得报告**：完成所有问题后，自动生成完整诊断报告

### 诊断流程

```
欢迎与初始化
  ↓
Q1: 选择资产类型（多选）
  ↓
逐类资产详细问答
  ├─ 境外银行存款 (8个问题)
  ├─ 境外证券投资 (5个问题)
  ├─ 离岸公司 (12个问题)
  ├─ 境外保单 (7个问题)
  └─ 离岸信托 (12个问题)
  ↓
风险评估
  ↓
生成诊断报告
  ├─ 诊断概要
  ├─ 资产总览
  ├─ 各资产信息披露义务判定
  ├─ CRS信息交换风险评估
  ├─ 合规建议
  └─ 附录（名词解释、参考资料）
  ↓
后续支持（回答疑问）
```

### 预计时间

- **简单情况**（1-2种资产类型）：10-15分钟
- **复杂情况**（3-5种资产类型）：20-30分钟
- **报告生成**：自动完成，约1-2分钟

## 技术架构

### 查询驱动（Query-Driven）架构

本Skill采用"查询驱动"架构，而非"记忆驱动"：

```
SKILL.md（核心指令）
    ↓ Read工具
reference/（数据文件）
    ├─ questions.md          # 每次提问前Read
    ├─ crs_jurisdictions.md  # 匹配国家时Read
    ├─ risk_assessment_rules.md  # 风险评估时Read
    ├─ validation_guide.md   # 校验答案时Read
    └─ report_template.md    # 生成报告时Read
```

**优势**：
- ✅ 准确性高（不依赖AI记忆，直接读取权威数据）
- ✅ 可维护性好（数据更新只需修改文件，不需重新训练）
- ✅ 可追溯性强（每个决策都有文件依据）
- ✅ 可扩展性好（增加新功能只需添加新文件）

### Read工具使用示例

```markdown
# 场景1: 提问前读取问题详情
Read {"path": "reference/questions.md"}
搜索: "## Q1-2"
提取: 问题文本、选项、跳转逻辑

# 场景2: 匹配CRS参与国
Read {"path": "reference/crs_jurisdictions.md"}
搜索: "香港" 或 "Hong Kong"
匹配: 中国香港（CRS参与国，2018年实施）

# 场景3: 应用风险评估规则
Read {"path": "reference/risk_assessment_rules.md"}
定位: "一、境外银行存款风险评估规则"
应用: 高/中/低风险判定条件
```

## 常见问题（FAQ）

### Q1: 这个Skill安全吗？我的信息会被泄露吗？

**A**: 完全安全。本Skill在本地运行，您的回答和诊断结果：
- ✅ 不会上传到任何服务器
- ✅ 不会被保存到云端
- ✅ 不会被第三方获取
- ✅ 仅在您的本地环境中处理

### Q2: 诊断报告的准确性如何？

**A**: 本Skill基于以下权威来源：
- OECD官方CRS规则和实施指南
- 中国《个人所得税法》《企业所得税法》
- 国家税务总局的CRS相关公告
- 专业税务机构的实务经验

但请注意：
- ⚠️ 本报告基于您提供的信息，如信息不准确，结论可能有偏差
- ⚠️ 税法和CRS政策可能变化，建议定期更新评估
- ⚠️ 复杂情况建议咨询专业税务顾问

### Q3: 使用这个Skill需要多少时间？

**A**: 取决于您涉及的资产类型：
- 1-2种资产类型：10-15分钟
- 3-5种资产类型：20-30分钟
- 报告生成：自动完成，1-2分钟

### Q4: 我可以中途暂停吗？

**A**: 可以。您可以随时暂停：
- 关闭对话窗口，Skill会保存您的进度（在当前会话中）
- 下次打开继续回答
- 如需重新开始，只需启动新的诊断会话

### Q5: 如果我不确定某个问题的答案怎么办？

**A**: Skill提供了灵活的处理方式：
- 可以选择"不确定"选项（如有）
- 可以跳过非必填问题
- Skill会采取保守评估，并在报告中提示您核实
- 可以稍后修改回答并重新生成报告

### Q6: 这个Skill会替代专业税务顾问吗？

**A**: 不会。本Skill的定位是：
- ✅ 帮助您初步了解CRS风险
- ✅ 识别需要关注的关键问题
- ✅ 提供基本的合规建议
- ✅ 帮助您准备咨询专业顾问时的问题清单

对于复杂情况（如高风险资产、CFC规则、架构优化等），仍然需要咨询专业税务顾问。

### Q7: CRS名单会更新吗？

**A**: 会的。CRS参与国名单会不断更新：
- 本Skill包含截至2026年3月的最新名单（100+国家/地区）
- 如需最新名单，请访问OECD官方网站
- 建议定期（每年）重新进行诊断，以获取最新评估

### Q8: 美国账户也属于CRS范围吗？

**A**: 不属于。美国不参与CRS，而是实施FATCA：
- 美国金融机构不会通过CRS向其他国家交换信息
- 但FATCA要求全球金融机构向美国IRS报告美国人的账户
- 本Skill会正确识别美国账户为"非CRS参与国"

### Q9: 我可以为多人使用这个Skill吗？

**A**: 可以。每次诊断是独立的：
- 适合为家庭成员、朋友等多人使用
- 每人的诊断结果互不影响
- 建议为每个人生成独立的诊断报告

### Q10: 发现问题或有改进建议怎么办？

**A**: 我们欢迎您的反馈：
- 报告问题：通过项目的Issue跟踪系统
- 改进建议：提交Pull Request或联系项目维护者
- 数据更新：如发现CRS名单或规则有误，请及时告知
