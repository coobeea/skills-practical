# CRS参与国/地区名单

## 文件说明

本文件包含截至2026年3月已承诺或正式实施CRS（Common Reporting Standard，共同申报准则）的国家和地区名单。

**数据来源**: OECD官方CRS参与国名单
**更新日期**: 2026年3月
**参与国总数**: 100+

## 使用指南（供大模型使用）

### 如何使用本文件

1. **国家/地区匹配**: 当用户提到某个国家/地区时,在本文件中搜索匹配
2. **模糊匹配**: 支持中文名、英文名、常见简称、别名匹配
3. **CRS状态判断**: 匹配成功=CRS参与国，未匹配=非CRS参与国或特殊地区
4. **风险评估**: CRS参与国的账户需要重点关注信息交换风险

### 匹配规则

- 优先匹配官方中文名
- 其次匹配英文名
- 支持常见简称（如"香港"匹配"中国香港"）
- 忽略大小写和多余空格
- 如果用户输入不明确，友好询问澄清

### 非CRS地区示例

以下地区**不是**CRS参与国（截至2026年3月）：
- **美国**（美国采用FATCA，不参与CRS）
- **台湾**
- **朝鲜**
- 部分加勒比海小国

## AI操作指引

### 匹配流程

```
步骤1: 接收用户输入的国家/地区名称
  ↓
步骤2: 标准化输入（去除空格、转换大小写）
  ↓
步骤3: 在本文件中搜索匹配
  ├─ 优先匹配中文名
  ├─ 其次匹配英文名
  └─ 最后匹配常见简称
  ↓
步骤4: 判断匹配结果
  ├─ 匹配成功 → CRS参与国
  │   └─ 输出: 该地区是CRS参与国，信息交换风险高
  │
  └─ 匹配失败 → 可能是非CRS参与国
      ├─ 检查是否为"美国"及其变体 → 特殊说明
      └─ 其他情况 → 友好询问用户澄清或查证
```
