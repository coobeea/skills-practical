# CRS 资产信息交换风险评估报告 — 格式模板

本文件定义评估报告的四章节结构和内容规范。SKILL.md 在生成最终报告时严格按本模板输出，不得增减章节，不得改变呈现顺序。

**格式规范：**
- 标有`[动态内容]`的位置，替换为本次评估的实际信息。
- 第二章每项资产独立呈现，有几项资产就呈现几项，不合并，不省略。
- 理由说明必须使用自然语言段落，禁止使用条目列表替代段落。
- 若某类资产用户回答"否"（不持有），该类资产不出现在报告中。

**CRS 资产信息交换风险评估报告**

**第一章 基本情况**

根据您提供的信息，您已确认为**中国内地税务居民**。本报告针对您持有的海外金融资产，在 CRS（共同申报准则）框架下评估信息是否可能被交换回中国内地税务机关。

本次评估基于您提供的信息和 CRS 通用规则，评估结论分为三类：**会被交换**（信息将通过 CRS 渠道传递至中国内地税务机关）、**不会被交换**（基于当前信息，不在 CRS 交换范围内或所在地与中国内地尚未激活交换关系）、**存在风险，建议进一步分析**（信息不完整或情况较为复杂，无法做出明确判断）。

**第四章 重要声明**

本报告基于您提供的信息及 CRS 通用规则生成，**不构成正式税务意见**。CRS 规则在各参与国的具体执行中可能存在差异，具体信息交换情况以相关国家和地区税务机关的实际执行为准。

本次评估所用 CRS 成员国与激活状态数据版本日期为 **[填写 crs-member-countries.md 的数据版本日期]**，该数据存在时效性，建议定期核实最新状态。

如您的资产情况发生变化（如新增资产、架构调整、税务居民身份变化等），或需要针对特定资产进行详细的税务合规规划，建议咨询专业税务顾问。

本报告仅覆盖 CRS 框架下的信息交换风险，不涵盖 FATCA（美国《海外账户税收合规法》）、BEPS（税基侵蚀和利润转移）等其他国际税务合规要求，如有相关需求请另行专项咨询。
