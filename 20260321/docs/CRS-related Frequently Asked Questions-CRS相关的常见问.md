### 与CRS相关的常见问题

> （2019年2月）
>
> 经合组织维护并定期更新有关通用报告标准（CRS）应用的常见问题清单（FAQ）。这些常见问题解答来自企业和政府代表。此类问题的答案进一步提高了CRS的准确性，并有助于确保实施的一致性。有关CRS的更多信息，请参见[[自动交换门户]{.underline}。](http://www.oecd.org/tax/automatic-exchange/common-reporting-standard/)
>
> *新的或更新的常见问题解答*

| **1.报告余额或价值**                                                                                                                                                                                                                          |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **如果金融机构不经常确定价值（例如，没有定期重新计算以向客户报告），应报告股权的余额或价值？**                                                                                                                                                |
|                                                                                                                                                                                                                                               |
| 该标准将在权益情况下的账户余额或价值定义为金融机构出于需要最频繁地确定价值的目的而计算出的价值（注释，第1节，A（4））。这个值是什么取决于具体事实。如果情况不同，例如，如果金融机构未以其他原因重新计算余额或价值，则可能是收购时的利息价值。 |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **2.汇总和排除的帐户**                                                                                                                                                                                                                        |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **应用汇总规则时是否需要包括排除帐户？**                                                                                                                                                                                                      |
|                                                                                                                                                                                                                                               |
| 否。汇总规则是指财务帐户的汇总（第七节，C节）。金融帐户的定义特别排除了排除帐户（第八节，C（1））。                                                                                                                                           |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **3.帐户持有人信息**                                                                                                                                                                                                                          |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **报告金融机构如何报告没有名字和姓氏的个人？**                                                                                                                                                                                                |
|                                                                                                                                                                                                                                               |
| CRS架构要求填写姓氏和名字的数据元素。如果个人的法定姓名是单名或单名，则姓氏数据元素应填写为" NFN"（无名），姓氏字段应使用帐户持有人的姓氏填写。                                                                                               |

| **4. 关于托管账户贷记或支付的销售收益的报告**                                                                                                                                                                                                      |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **第一节A（5）（b）项规定，在托管账户的情况下，应报告出售或赎回已支付或贷记到该账户的金融资产的总收益总额。**                                                                                                                                      |
|                                                                                                                                                                                                                                                    |
| **当就托管账户付款或贷记这些总收益时，是否还需要报告这些总收益？**                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                    |
| 是的，与第一节A（5）（a）分段所列的收入项目一样，报告根据A（5）（b）分段出售或赎回托管账户中金融资产的总收益如果这些总收入已支付或贷记到该帐户，以及该款项已被支付或贷记到该帐户，则需要第一节）。                                                 |
|                                                                                                                                                                                                                                                    |
| 如果金融资产存放在托管账户中，则维护该托管账户的托管机构应报告出售或赎回此类金融资产的任何收入和总收入，而不论该账户向哪个账户支付或记                                                                                                             |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **5.收集TIN的要求**                                                                                                                                                                                                                                |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **关于第一节的评注第30段规定，对于未由其发行TIN的应报告人持有的应报告帐户，不需要报告TIN。如果金融机构要求应报告人获得并提供TIN，则该应报告人在其居住辖区有资格或可能有资格获得TIN（或功能等效物），但无需获得TIN和还没有获得TIN？**               |
|                                                                                                                                                                                                                                                    |
| 没有。                                                                                                                                                                                                                                             |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **6.间歇性分配给作为报告金融机构的信托的全权受益人**                                                                                                                                                                                               |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **如果是金融机构的信托，则股权被视为是全部或部分信托的委托人或受益人的任何人持有。出于这些目的，如果受益人仅在日历年或其他适当的报告期内获得分配，则仅可从信托获得全权分配的受益人将被视为信托的受益人（请参阅第VIII（C）（4）节）及相关评论）。** |
|                                                                                                                                                                                                                                                    |
| **如果作为金融机构的信托的全权受益人在给定年份而不是在下一年从该信托获得分配，那么该次年没有分配的情况是否应视为帐户关闭？**                                                                                                                       |
|                                                                                                                                                                                                                                                    |
| 不可以，只要没有将受益人永久性地排除在信托的未来分配范围之外，那么没有分配将不会导致帐户关闭。                                                                                                                                                     |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **7.报告作为实体的定居者的控制人**                                                                                                                                                                                                                 |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **该标准规定，在信托的设立人是实体的情况下，报告金融机构还必须确定该设立人的控制人并将其报告为控制人。是否仅在结算年度或随后的几年中识别并报告控制人？**                                                                                           |
|                                                                                                                                                                                                                                                    |
| 不仅在结案年度，而且在随后的所有年份中，都需要确定和报告财产授予人的控制人。                                                                                                                                                                       |

|                                                                                                                                                                                                                                                                                                                                                              |
| a)  **由报告金融机构的信托所维护？**                                                                                                                                                                                                                                                                                                                         |
|                                                                                                                                                                                                                                                                                                                                                              |
| b)  **由报告金融机构维护的被动NFE信托？**                                                                                                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                                                              |
| 在这两种情况下，要报告的财务活动都包括帐户关闭的事实和在相关报告期内向帐户持有人支付的总付款。                                                                                                                                                                                                                                                               |
+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **9.从不是应报告管辖权的控制人那里收集TIN**                                                                                                                                                                                                                                                                                                                  |
+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **根据第VIII（D）（8）节，第VIII（A）（6）（b）节所述的非参与管辖金融机构的投资实体为被动NFE，且第V节或第C节中的尽职调查程序必须将第VI节应用于投资实体的帐户，以确定其帐户是否为可报告帐户。如果被动NFE具有一个或多个控制人（应报告人），则该帐户为应报告人。如果控制人不是应报告的管辖权人，是否需要收集该控制人的TIN？**                                   |
|                                                                                                                                                                                                                                                                                                                                                              |
| 遵守国内法的规定，特别是关于标准附件5中反映的所谓"更广泛的方法"的规定，如果控制人不是应报告的管辖权人，则不需要从这样的控制人。                                                                                                                                                                                                                              |

| **10.用于CRS的使用权资格**                                                                                                                                                                                                                                                                                                                                   |
+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **为了CRS目的，如何处理使用权（使用和从财产中获取利润的合法权利）？**                                                                                                                                                                                                                                                                                        |
|                                                                                                                                                                                                                                                                                                                                                              |
| 出于尽职调查和报告目的，裸所有者（"nu-propriétaire"）和使用权（"usufruitier"）均可被视为联合账户持有人或信托的控制人。                                                                                                                                                                                                                                       |
+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **11.正在清算中的报告金融机构的报告义务**                                                                                                                                                                                                                                                                                                                    |
+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **正在清算或清算中的报告金融机构应如何履行CRS规定的尽职调查和报告义务？**                                                                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                                                              |
| 作为一般规则，根据尽职调查程序（第II（A）节），从确定金融账户之日起，将其视为应报告账户。可报告帐户一直保持可报告状态，直到其不再是可报告帐户为止（例如，由于该帐户已关闭）。如果由于报告金融机构的清算或清算而关闭了应报告账户，则与该账户有关的信息将保持每年可报告的状态，直到报告金融机构关闭该金融账户之日（第II（A）节的注释）清算或清算框架内的机构。 |
|                                                                                                                                                                                                                                                                                                                                                              |
| 在这方面，司法管辖区可以考虑到相关的国内法律规定，特别是在公司法和破产法方面，向清算或清算过程中如何履行其尽职调查和报告义务的行为提供进一步的指导。 。                                                                                                                                                                                                      |
|                                                                                                                                                                                                                                                                                                                                                              |
| 在这方面，一种选择是允许依赖第三方服务提供商，以确保报告金融机构的所有尽职调查和报告义务得到充分履行（第II（D）节）。                                                                                                                                                                                                                                        |

| ##### 第II-VII节：尽职调查要求                                                                                                                                                                                                                                                                                        |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **1.书面证明**                                                                                                                                                                                                                                                                                                        |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **该标准是否要求报告金融机构保留作为其尽职调查程序一部分而收集的书面证据的纸质副本？**                                                                                                                                                                                                                                |
|                                                                                                                                                                                                                                                                                                                       |
| 不需要。报告金融机构不需要保留书面证据的纸质副本，但可以这样做（关于第八节的评注第157段）。报告财务机构可以保留单据证据的原始，经核证的副本或影印本，或保留经审查的文档类型的注释，文档的审查日期以及文档的标识号（如果有）（例如， ，护照号码）。                                                                    |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **2.居住地址测试--要求人工审核文件证据**                                                                                                                                                                                                                                                                              |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **《标准》中要求用文件证据确认居住地址的要求是否要求对帐户进行人工审核？**                                                                                                                                                                                                                                            |
|                                                                                                                                                                                                                                                                                                                       |
| 该标准不需要纸质检查即可检查文件证据。通常，居住地址测试的要求是居住地址基于证件证据（第III节，B，（1）和相关的注释）。如果金融机构保留了如上所述的证明文件，或者制定了政策和程序来确保当前居住地址与所提供的证明文件上的地址相同，则报告金融机构将拥有满足居住地址测试的文件证据要求。                               |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **3.居住地址测试--两个居住地址**                                                                                                                                                                                                                                                                                      |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **在进行居住地址测试后，是否有可能确定帐户持有人有两个居住地址？**                                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                       |
| 是。只要满足所有适用于居住地址测试的条件（第III节，B，（1）和相关注释），那么居住地址测试就有可能找到两个地址。例如，对于在国家A中维护的银行帐户，在国家B的居民在国家B和国家C工作且有一半时间的情况下，银行可以拥有两个满足要求的地址。在这种情况下，可以寻求自我证明，也可以将帐户报告给所有有居住地址的应报告辖区。 |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **4.依靠AML / KYC程序来识别控制人**                                                                                                                                                                                                                                                                                   |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **对于帐户余额或总值不超过1,000,000美元的现有实体帐户，如果金融机构持有有关控制人姓名的信息而没有其他信息，则尽职调查和报告要求是什么？是否不需要按照适用的AML / KYC程序收集此类信息？**                                                                                                                              |
|                                                                                                                                                                                                                                                                                                                       |
| 该标准规定，对于余额或价值低于100万美元的帐户（应用汇总规则后），金融机构可依赖出于监管或客户关系目的而收集和维护的信息，包括AML /                                                                                                                                                                                    |
| KYC程序以确定是否控制人是应报告人（第V，D，（2），c）节）。由于在给定的示例中，金融机构没有且不需要归档任何表明控制人可能是应报告人的信息，因此它无法记录控制人的住所，也无需报告该人为控制人。                                                                                                                       |

|                                                                                                                                                                                                                  |
| 否。所有权实体中的中间实体的CRS状态与这些目的无关。                                                                                                                                                              |

| **6.针对CRS的AML / KYC程序和尽职调查**                                                                                                                                                                           |
+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **关于第III-VII节中规定的尽职调查程序，金融机构将对AML / KYC程序进行更改会产生什么后果？**                                                                                                                       |
|                                                                                                                                                                                                                  |
| 第VIII（E）（2）节规定，术语"AML / KYC程序" 是指根据报告金融机构所遵循的反洗钱或类似要求对报告金融机构进行的客户尽职调查程序。因此，为了执行第III-VII节的尽职调查程序，适用的AML /                               |
| KYC程序是金融机构在给定的时间要接受的程序，只要对于新帐户，该程序与2012年FATF建议。                                                                                                                              |
|                                                                                                                                                                                                                  |
| 如果对适用的AML / KYC程序进行了修订（例如，在执行新的FATF建议的司法管辖区中），则金融机构可能会被要求收集和维护该司法管辖区中用于AML /                                                                           |
| KYC的其他信息。出于第III-VII节规定的尽职调查程序的目的，并根据关于第III节的评注第17段，必须使用根据经修订的AML /                                                                                                 |
| KYC程序获得的其他信息来确定是否进行了更改与帐户持有人和/或控制人的身份和/或可报告状态有关的情况。                                                                                                                |
|                                                                                                                                                                                                                  |
| 如关于第七节的评注第4段所述，如果所获得的额外信息与某人通过自我证明提出的主张不一致，则情况有所变化，金融机构将有理由知道自我认证不可靠或不正确。                                                                |
+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **7.金融机构建立税务居民的义务**                                                                                                                                                                                 |
+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **根据新帐户程序，金融机构标准在建立其客户的税收居所方面有哪些义务？**                                                                                                                                           |
|                                                                                                                                                                                                                  |
| 不需要金融机构向客户提供税务建议或进行法律分析以确定自我认证的合理性。相反，按照该标准的规定，对于新帐户，金融机构可以依靠客户做出的自我认证，除非它知道或有理由知道该自我认证是不正确或不可靠的（"合理性"测试） |
| ，该信息将基于与开设帐户有关的信息，包括根据AML / KYC程序获得的任何文件。该标准提供了合理性测试的应用示例（第四节，A节和相关注释）。                                                                             |
|                                                                                                                                                                                                                  |
| 该标准还指出，预期参与的司法管辖区将有助于纳税人确定其住所并向其提供有关税收的信息（第四节评注的第6段和第六节评注的第9段）                                                                                       |
| 。经合组织正在通过信息的集中分发（在自动交换门户上）促进这一进程。金融机构还可以引导客户使用此信息。                                                                                                             |

| **8.TIN的验证**                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **关于自我证明上提供的纳税人识别号（TIN），报告金融机构何时会知道或有理由知道自我证明不正确或不可靠？**                                                                                                                                                                                                                                                                                                                                                              |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 该标准规定，除非报告金融机构知道或有理由知道该自我认证是不正确或不可靠的，否则它可以依靠自我认证（第七节，A段和相关的评注）。除提供有关自我认证的其他信息外，这还包括与应报告管辖权有关的TIN。该标准期望参与管辖区将为报告金融机构提供有关TIN的发行，收集以及尽可能的实际结构和其他规范的信息（对第八节的注释，第149段）。经合组织将通过信息的集中分发（在自动交换门户上）促进这一进程。                                                                             |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 如果自认证不包含TIN，并且自动交易门户网站上包含的信息表明应报告司法管辖区向所有纳税居民签发TIN，则报告金融机构将有理由知道自认证不可靠或不正确。该标准不需要报告金融机构使用自动交易门户上提供的信息来确认TIN的格式和其他规格。但是，报告金融机构可能还是希望这样做，以提高收集到的信息的质量，并最大程度地减少与报告不正确的TIN有关的任何跟进工作的行政负担。在这种情况下，他们还可以使用提供TIN检查模块的区域和国家网站，以进一步验证自我认证中提供的TIN的准确性。 |

| **9.自我认证--表示"肯定"**                                                                                                                                                                                                                                                                                                                                                                                                                                           |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **根据本标准开立的自我认证要求有效，即必须由客户签署或积极确认（关于第四节的评注第7段）。应该如何理解"否则得到肯定"？**                                                                                                                                                                                                                                                                                                                                              |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 如果进行自我认证的人向金融机构明确表示他们同意通过自我认证所作的陈述，则可以对自我认证予以肯定。在所有情况下，金融机构都希望能够以肯定的方式证明自我认证得到肯定肯定（例如，录音，数字足迹等）。金融机构用于获得自我认证的方法应与金融机构开立帐户所遵循的程序相一致。除了自我认证本身以外，金融机构还需要保留此过程的记录以用于审核。                                                                                                                               |

| **10.口头自我认证**                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

| **1.间接所有权时的相关实体定义**                                                                                                                                                                                                                                                                                                                       |
+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **为了确定一个实体是否与另一个实体相关，根据标准第VIII节E（4）小节，需要验证一个实体是否控制另一个实体或两个实体是否处于共同控制之下。该条款还规定，控制权包括实体中50％以上的投票和价值的直接或间接所有权。**                                                                                                                                         |
|                                                                                                                                                                                                                                                                                                                                                        |
| **如果间接拥有一个实体在另一个实体中的投票权和价值，该所有权是否应按比例衡量？**                                                                                                                                                                                                                                                                       |
|                                                                                                                                                                                                                                                                                                                                                        |
| ***例：***                                                                                                                                                                                                                                                                                                                                             |
|                                                                                                                                                                                                                                                                                                                                                        |
| ***实体A拥有实体B的总投票权51％和总价值的51％。实体B依次拥有实体C的总投票权51％和总价值的51％ 。实体A和实体C相关的实体吗？在比例规则下，实体将不相关，因为该规则将要求表决权和价值的百分比相乘，而实际上，实体A有效地控制了实体C。***                                                                                                                  |
|                                                                                                                                                                                                                                                                                                                                                        |
| 如果一个实体通过一个或多个共同母公司实体通过一个或多个所有权链相连，并且该共同母公司直接拥有至少一个其他实体中50％以上的股权或其他股权，则这些实体被视为相关实体。所有权链应理解为一个或多个实体所拥有的所有权超过实体股票总投票权的50％，并超过实体股票总价值的50％。                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                        |
| *[例：]{.underline}*                                                                                                                                                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                        |
| *根据第VIII节E（4）的规定，实体A和C被视为"相关实体"，因为实体A的直接所有权超过了股票总投票权的50％，超过了股票总投票权的50％。实体B的股票，并且由于实体B拥有股票总投票权的50％以上，实体C的股票总价值的50％以上的直接所有权。因此，实体A和C之间存在关联通过所有权链。尽管实体A按比例仅拥有实体C的股票和投票权总价值的26％，但实体A和实体C是关联实体。* |
