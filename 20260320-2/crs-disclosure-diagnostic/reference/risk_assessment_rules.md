# CRS风险评估规则

## 文件说明

本文件定义CRS信息披露诊断的风险评估规则和判定逻辑。

**版本**: v1.0
**用途**: 供大模型根据用户回答判断CRS风险等级和披露义务

## 使用指南（供大模型使用）

### 风险评估流程

```
步骤1: 收集用户回答的所有信息
  ↓
步骤2: 对每个资产类型应用对应的风险评估规则
  ↓
步骤3: 判定每个资产的风险等级（高/中/低）
  ↓
步骤4: 综合所有资产，确定总体风险等级（取最高值）
  ↓
步骤5: 生成风险评估报告
```

### 风险等级定义

| 风险等级 | 图标 | 含义 | 后果 |
|---------|------|------|------|
| 高风险 | 🔴 | 存在明确的CRS信息披露义务，信息已交换或即将交换至中国税务机关 | 账户信息将被交换，需主动申报并准备应对税务检查 |
| 中风险 | 🟡 | 可能存在CRS信息披露义务，需进一步核实或存在不确定性 | 建议核实账户状态，准备相关材料，可能需要补充申报 |
| 低风险 | 🟢 | 无CRS信息披露义务，或不涉及中国税务机关 | 暂无CRS风险，但需关注政策变化和其他税务义务 |

## 一、境外银行存款风险评估规则

### 1.1 高风险判定条件（🔴）

**必须同时满足以下所有条件**:

```
条件1: 用户是中国税收居民（Q1-2-1 = "是中国税收居民"）
AND
条件2: 账户开立地是CRS参与国/地区（Q1-1 匹配CRS名单成功）
AND
条件3: 满足以下任一子条件：
  - 账户余额 >= 100万美元（Q1-3-2 = "100万至250万美元" OR "超过250万美元"）
  OR
  - 账户持有人是本人独有或联名（Q1-2 = "本人独有" OR "与配偶/家人联名"）
AND
条件4: 非特殊类型账户（Q1-4 = "不是特殊类型账户"）
```

**风险说明**:
- 账户信息将通过CRS交换至中国税务机关
- 需主动申报境外账户信息（虽然目前无强制要求，但建议主动申报）
- 高余额账户（>100万美元）是税务机关重点关注对象

### 1.2 中风险判定条件（🟡）

**满足以下情况之一**:

```
情况1: 
  用户是中国税收居民
  AND 账户在CRS参与国
  AND 账户余额 < 100万美元（Q1-3-2 = "低于100万美元"）

情况2:
  用户是中国税收居民
  AND 账户在CRS参与国
  AND 账户由他人名义持有但用户是实际受益人（Q1-2 = "由他人持有" AND Q1-2-3 = "是该账户资金的实际提供者或最终受益人"）

情况3:
  用户是中国税收居民
  AND 账户在CRS参与国
  AND 不确定是否为特殊类型账户（Q1-4 = "不确定是否为特殊类型账户"）
```

**风险说明**:
- 可能需要披露，建议核实账户状态
- 虽然余额较低，但仍在CRS申报范围内
- 代持账户如实际控制，仍可能被识别

### 1.3 低风险判定条件（🟢）

**满足以下情况之一**:

```
情况1: 账户在非CRS参与国（如美国）（Q1-1 匹配CRS名单失败，且确认为美国等非参与国）

情况2: 用户不是中国税收居民（Q1-2-1 = "不是中国税收居民"）

情况3: 账户属于特殊类型（Q1-4 = "是特殊类型账户"）
  说明: 退休金账户、学生账户等可能享受CRS豁免
```

**风险说明**:
- 暂无CRS信息交换风险
- 但需注意其他税务义务（如FATCA、当地税务申报）

## 二、境外证券投资风险评估规则

### 2.1 高风险判定条件（🔴）

**必须同时满足以下所有条件**:

```
条件1: 实际控制人是中国税收居民
  - 个人持有: Q2-1-2-1 = "是中国的税收居民"
  - 公司/信托持有: Q2-1-2-2 = "是中国的税收居民"（实际控制人）
AND
条件2: 账户开立地是CRS参与国/地区（Q2-1-1 匹配CRS名单成功）
AND
条件3: 投资类型包含高风险产品（Q2-2 包含"私募基金/对冲基金" OR "衍生品"）
```

**风险说明**:
- 证券账户信息将通过CRS交换
- 投资收益需在中国申报纳税
- 私募基金和对冲基金可能涉及复杂的信息穿透

### 2.2 中风险判定条件（🟡）

**满足以下情况**:

```
条件1: 实际控制人是中国税收居民
AND
条件2: 账户在CRS参与国
AND
条件3: 投资类型仅包含常规产品（Q2-2 = "上市公司股票" OR "债券" OR "公募基金"）
```

**风险说明**:
- 账户信息将被交换，但投资类型相对简单
- 投资收益需申报，但计算相对明确
- 建议保留交易记录和税务凭证

### 2.3 低风险判定条件（🟢）

**满足以下情况之一**:

```
情况1: 账户在非CRS参与国（Q2-1-1 匹配CRS名单失败）

情况2: 实际控制人不是中国税收居民
```

**风险说明**:
- 暂无CRS信息交换风险
- 但需关注当地税务申报要求

## 三、离岸公司风险评估规则

### 3.1 高风险判定条件（🔴）

**必须同时满足以下所有条件**:

```
条件1: 用户是中国税收居民（Q3-3-2 = "是中国税收居民"）
AND
条件2: 用户是公司的控制人
  - 股权比例 > 25%（从Q3-3-1提取）
  OR 对公司有实际控制影响力
AND
条件3: 公司注册地是CRS参与国（Q3-1-1 匹配CRS名单成功）
AND
条件4: 满足以下任一子条件：
  - 公司是Passive NFE（消极非金融机构）：
    Q3-2-2 = "否，主动经营业务收入比例和资产比例均未超50%"
  OR
  - 公司开立了境外金融账户：
    Q3-3-4 = "是，在境外开立有金融账户"
```

**风险说明**:
- **Passive NFE的控制人信息将被穿透识别和交换**
- 如公司有境外金融账户，账户信息将被报送
- 可能需要申报受控外国企业（CFC）
- 公司利润可能被视为个人应税收入

### 3.2 中风险判定条件（🟡）

**满足以下情况**:

```
情况1: 
  用户是中国税收居民
  AND 用户是控制人
  AND 公司在CRS参与国
  AND 公司是Active NFE（积极非金融机构）：
    Q3-2-2 = "是，主动经营业务收入比例超50%或主动经营业务资产比例超50%"

情况2:
  用户是中国税收居民
  AND 用户股权比例 <= 25% 但有部分影响力
  AND 公司在CRS参与国

情况3:
  用户是中国税收居民
  AND 用户是控制人
  AND 公司在CRS参与国
  AND 不确定主动经营业务比例（Q3-2-2 = "不清楚..."）
```

**风险说明**:
- Active NFE相对风险较低，但仍需关注
- 建议保留公司经营活动证明材料
- 需评估是否触发CFC规则

### 3.3 低风险判定条件（🟢）

**满足以下情况之一**:

```
情况1: 公司注册地是非CRS参与国（Q3-1-1 匹配CRS名单失败）

情况2: 用户不是中国税收居民（Q3-3-2 = "不是中国税收居民"）

情况3: 用户既不是控制人也无控制影响力（Q3-3-1 股权<10% 且无控制权）

情况4: 公司是金融机构（Q3-1-2 = "是金融机构"）
  说明: 金融机构本身是CRS报送主体，不需要穿透
```

**风险说明**:
- 暂无CRS控制人识别风险
- 但需关注其他税务义务（如分红税、资本利得税）

## 四、境外保单风险评估规则

### 4.1 高风险判定条件（🔴）

**必须同时满足以下所有条件**:

```
条件1: 保单有现金价值或是年金（Q4-1-1 = "是，有现金价值/是年金"）
AND
条件2: 保单持有人是中国税收居民（Q4-2-1-1 = "是中国税收居民"）
AND
条件3: 保单开立地是CRS参与国（推断：主要离岸保险中心如香港、新加坡等均为CRS参与国）
AND
条件4: 满足以下任一子条件：
  - 持有人是本人（Q4-2-1 = "本人"）
  OR
  - 持有人是家族信托（Q4-2-1 = "家族信托"），且受益人是中国税收居民
```

**风险说明**:
- 保单账户信息将通过CRS交换
- 保单现金价值和分红需申报
- 年金领取收入需在中国纳税

### 4.2 中风险判定条件（🟡）

**满足以下情况**:

```
情况1:
  保单有现金价值
  AND 持有人是中国税收居民
  AND 持有人是配偶/子女/父母（Q4-2-1 = "配偶" OR "子女" OR "父母"）

情况2:
  保单有现金价值
  AND 持有人不是中国税收居民
  BUT 指定受益人是中国税收居民（Q4-2-3-1 = "该受益人是中国税收居民"）
```

**风险说明**:
- 如持有人是家人，可能涉及赠与税或遗产税问题
- 受益人是中国税收居民，未来领取保险金时可能需要纳税
- 建议保留保单文件和缴费记录

### 4.3 低风险判定条件（🟢）

**满足以下情况之一**:

```
情况1: 保单是纯消费型（Q4-1-1 = "否，是纯消费型保障"）
  说明: 纯消费型保单不属于CRS申报范围

情况2: 持有人和受益人均不是中国税收居民

情况3: 受益人按法定继承人或不确定（Q4-2-3 = "未指定，按法定继承人" OR "不确定"）
  且持有人不是中国税收居民
```

**风险说明**:
- 暂无CRS风险
- 但需关注保单赔付时的税务处理

## 五、离岸信托风险评估规则

### 5.1 高风险判定条件（🔴）

**必须同时满足以下所有条件**:

```
条件1: 用户是中国税收居民（Q5-1-2 = "是中国税收居民"）
AND
条件2: 信托开立了境外金融账户（Q5-4-1 = "是，有开立境外金融账户"）
AND
条件3: 信托账户开立地推断为CRS参与国（根据常见信托中心判断）
AND
条件4: 满足以下任一子条件：

  子条件A（委托人保留权力）:
    用户是委托人（Q5-1-1 包含"委托人"）
    AND 保留了对信托的权力（Q5-2-A-2 = "是，保留了权力"）
    
  OR
  
  子条件B（固定受益人且定期分配）:
    用户是受益人（Q5-1-1 包含"受益人"）
    AND 是固定受益人（Q5-2-B-1 = "明确指名的固定受益人"）
    AND 定期获得分配（Q5-2-B-2 = "是"）
    
  OR
  
  子条件C（保护人有控制权）:
    用户是保护人（Q5-1-1 包含"保护人"）
    AND 拥有控制权（Q5-2-C-1 = "是，拥有对受托人的指令权..."）

AND
条件5: 信托属于投资实体（Q5-3-1 = "是，它是一个投资工具..."）
```

**风险说明**:
- **信托属于投资实体，需穿透至委托人/受益人层面**
- 保留权力的委托人：信托可能被认定为"未生效"，资产视为个人资产
- 固定受益人且定期分配：分配收入需在中国申报纳税
- 保护人有控制权：可能被认定为实际控制人
- 信托账户信息将通过CRS交换至中国税务机关

### 5.2 中风险判定条件（🟡）

**满足以下情况**:

```
情况1:
  用户是中国税收居民
  AND 用户是潜在受益人（Q5-2-B-1 = "属于潜在受益人范围"）
  AND 信托有境外金融账户
  AND 信托是投资实体
  
  说明: 潜在受益人未来可能获得分配，需关注

情况2:
  用户是中国税收居民
  AND 用户是委托人但未保留权力（Q5-2-A-2 = "否，未保留任何权力"）
  AND 信托有境外金融账户
  AND 信托是投资实体
  
  说明: 虽未保留权力，但仍可能被识别为信托设立人

情况3:
  用户是中国税收居民
  AND 用户是控制人（委托人/保护人）
  AND 信托主要持有并经营公司（Q5-3-1 = "否，它主要持有并经营一家公司"）
  
  说明: 经营型信托相对风险较低，但仍需关注

情况4:
  用户是中国税收居民
  AND 用户是控制人
  AND 不确定信托活动类型（Q5-3-1 = "不确定信托的主要活动..."）
  
  说明: 需核实信托分类
```

**风险说明**:
- 可能需要披露信托信息
- 建议保留信托文件和分配记录
- 咨询专业税务顾问评估信托架构

### 5.3 低风险判定条件（🟢）

**满足以下情况之一**:

```
情况1: 用户不是中国税收居民（Q5-1-2 = "不是中国税收居民"）

情况2: 信托未开立境外金融账户（Q5-4-1 = "否，未开立任何境外金融账户"）
  说明: 无金融账户则无CRS申报义务

情况3: 信托主要持有非金融资产（Q5-3-1 = "否，它主要持有非金融资产"）
  AND 用户仅是受益人且非固定受益人
  
  说明: 非金融资产持有型信托，相对风险较低
```

**风险说明**:
- 暂无CRS信息交换风险
- 但需关注信托设立和资产转让的税务处理
- 关注未来分配时的纳税义务

## 六、综合风险等级判定规则

### 6.1 总体风险等级判定

**规则**: 取所有涉及资产类型的最高风险等级

```
IF 任一资产类型 = 🔴 高风险
  THEN 总体风险等级 = 🔴 高风险

ELSE IF 任一资产类型 = 🟡 中风险
  THEN 总体风险等级 = 🟡 中风险

ELSE IF 所有资产类型 = 🟢 低风险 OR 未涉及
  THEN 总体风险等级 = 🟢 低风险
```

### 6.2 关键风险点识别

**识别原则**: 列举所有已识别的高风险和中风险因素

**高风险因素**（按优先级排序）:
1. **消极非金融机构(Passive NFE)的控制人**
   - 条件: 离岸公司 + Passive NFE + 用户是控制人 + 中国税收居民
   - 风险: 控制人信息将被穿透识别和交换

2. **投资型信托的委托人/受益人**
   - 条件: 离岸信托 + 投资实体 + 委托人保留权力/固定受益人 + 开立境外账户
   - 风险: 信托账户将穿透至委托人/受益人，信息被交换

3. **高余额境外账户**
   - 条件: 境外银行存款/证券账户 + 余额>100万美元 + 中国税收居民
   - 风险: 高余额账户是税务机关重点关注对象

4. **有现金价值的境外保单**
   - 条件: 境外保单 + 有现金价值 + 持有人是中国税收居民
   - 风险: 保单账户信息被交换，现金价值和分红需申报

5. **私募基金/对冲基金投资**
   - 条件: 境外证券投资 + 投资私募基金/对冲基金 + 中国税收居民
   - 风险: 复杂产品的信息穿透，收益申报复杂

**中风险因素**:
1. 低余额但在CRS参与国的账户
2. Active NFE但开立境外账户的离岸公司
3. 潜在受益人身份的离岸信托
4. 代持但用户是实际受益人的账户

### 6.3 风险评分模型（可选）

**评分规则**（供参考）:

| 风险因素 | 分值 |
|---------|-----|
| 高余额账户（>250万美元） | +10 |
| 高余额账户（100-250万美元） | +7 |
| Passive NFE控制人 | +10 |
| 投资型信托委托人（保留权力） | +10 |
| 投资型信托固定受益人（定期分配） | +8 |
| 有现金价值的境外保单 | +6 |
| 私募基金/对冲基金投资 | +5 |
| Active NFE控制人 | +3 |
| 低余额账户（<100万美元） | +2 |

**风险等级判定**:
- 总分 >= 10: 🔴 高风险
- 总分 5-9: 🟡 中风险
- 总分 < 5: 🟢 低风险

## 七、特殊情况处理规则

### 7.1 不确定信息的处理

**原则**: 当用户回答"不确定"时，采取保守评估原则（往高风险方向判定）

**示例**:
- Q1-4 "不确定是否为特殊类型账户" → 不视为豁免账户，继续风险评估
- Q3-2-2 "不清楚主动经营业务的收入或资产比例" → 暂时视为Passive NFE，提示用户核实
- Q5-3-1 "不确定信托的主要活动..." → 建议用户咨询专业机构，暂时按中风险处理

### 7.2 非CRS参与国的特殊说明

**美国账户**:
- 虽不参与CRS，但美国实施FATCA
- 如用户是美国公民或绿卡持有者，需向IRS申报全球收入和资产
- 中国税收居民在美国的账户，不会通过CRS交换至中国，但需关注FATCA

**其他非CRS参与国**:
- 暂无信息交换风险
- 但需关注当地税务申报要求
- 关注未来可能加入CRS的政策变化

### 7.3 多重身份的处理

**场景**: 用户同时是委托人、受益人、保护人等多重角色

**处理规则**:
- 分别评估每个角色的风险
- 取最高风险等级
- 在报告中分别说明每个角色的风险

## 八、AI操作指引

### 8.1 风险评估步骤

```
步骤1: 读取用户所有回答,整理关键信息
  - 税收居民身份
  - 账户/公司/信托的开立地/注册地
  - 账户余额/股权比例/角色
  - 资产类型/账户类型/公司性质/信托分类

步骤2: 对每个资产类型,应用对应的风险评估规则
  - 匹配高/中/低风险判定条件
  - 记录匹配结果和理由

步骤3: 判定每个资产的风险等级
  - 输出: 🔴 高风险 / 🟡 中风险 / 🟢 低风险

步骤4: 综合判定总体风险等级
  - 取所有资产的最高风险等级

步骤5: 识别关键风险点
  - 列举所有高风险因素
  - 列举重要的中风险因素

步骤6: 生成风险评估报告
  - 填充report_template.md中的风险评估部分
```

### 8.2 判定逻辑示例

**示例1: 境外银行存款风险判定**

```python
# 伪代码示例
def assess_bank_account_risk(user_answers):
    # 提取关键信息
    is_china_resident = user_answers['Q1-2-1'] == '是中国税收居民'
    account_country = user_answers['Q1-1']
    is_crs_country = check_crs_jurisdiction(account_country)
    balance_range = user_answers['Q1-3-2']
    holder_type = user_answers['Q1-2']
    is_special = user_answers['Q1-4']
    
    # 高风险判定
    if (is_china_resident 
        and is_crs_country 
        and (balance_range in ['100万至250万美元', '超过250万美元'] 
             or holder_type in ['本人独有', '与配偶/家人联名'])
        and is_special == '不是特殊类型账户'):
        return '🔴 高风险', '满足所有高风险条件'
    
    # 中风险判定
    elif (is_china_resident 
          and is_crs_country 
          and balance_range == '低于100万美元'):
        return '🟡 中风险', '账户在CRS参与国但余额较低'
    
    # 低风险判定
    else:
        return '🟢 低风险', '不满足高/中风险条件'
```

## 九、风险评估决策树

### 9.1 境外银行存款决策树

```
是否是中国税收居民?
├─ 是 → 账户在CRS参与国?
│   ├─ 是 → 账户余额范围?
│   │   ├─ >= 100万美元 → 🔴 高风险
│   │   └─ < 100万美元 → 🟡 中风险
│   └─ 否 → 🟢 低风险
└─ 否 → 🟢 低风险
```

### 9.2 离岸公司决策树

```
是否是中国税收居民?
├─ 是 → 是否是控制人?
│   ├─ 是 → 公司在CRS参与国?
│   │   ├─ 是 → 公司是Passive NFE?
│   │   │   ├─ 是 → 🔴 高风险
│   │   │   ├─ 否(Active NFE) → 🟡 中风险
│   │   │   └─ 不确定 → 🟡 中风险(建议核实)
│   │   └─ 否 → 🟢 低风险
│   └─ 否 → 🟢 低风险
└─ 否 → 🟢 低风险
```

### 9.3 离岸信托决策树

```
是否是中国税收居民?
├─ 是 → 信托开立境外账户?
│   ├─ 是 → 信托是投资实体?
│   │   ├─ 是 → 用户是委托人(保留权力)或固定受益人(定期分配)?
│   │   │   ├─ 是 → 🔴 高风险
│   │   │   └─ 否 → 🟡 中风险
│   │   ├─ 否(Active/Passive NFE) → 🟡 中风险
│   │   └─ 不确定 → 🟡 中风险(建议核实)
│   └─ 否 → 🟢 低风险
└─ 否 → 🟢 低风险
```
