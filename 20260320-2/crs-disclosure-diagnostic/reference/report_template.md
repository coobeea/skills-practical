# CRS诊断报告生成模板

## 文件说明

本文件定义CRS信息披露诊断报告的生成模板和填充规则。

**版本**: v1.0
**用途**: 供大模型在诊断完成后生成结构化的Markdown报告

## 使用指南（供大模型使用）

### 报告生成流程

```
步骤1: 收集整理所有用户回答和诊断结果
  ↓
步骤2: 应用风险评估规则，确定每个资产的风险等级
  ↓
步骤3: 根据模板结构逐部分生成报告内容
  ↓
步骤4: 填充所有占位符 {variable}
  ↓
步骤5: 生成完整的Markdown格式报告
  ↓
步骤6: 输出给用户
```

### 报告结构

报告共包含6个主要部分：

1. **诊断概要** - 基本信息和资产类型总览
2. **资产总览** - 各资产类型的涉及情况和风险等级
3. **各资产信息披露义务判定** - 每个资产的详细分析（最重要部分）
4. **CRS信息交换风险评估** - 综合风险评估和关键风险点
5. **合规建议** - 针对性的行动建议
6. **附录** - 名词解释和参考资料

## 详细填充规则

### 第一部分：诊断概要

#### {timestamp}
- **来源**: 系统当前时间
- **格式**: `YYYY年MM月DD日 HH:MM`
- **示例**: `2026年03月20日 15:30`

#### {asset_types}
- **来源**: Q1用户选择的资产类型
- **格式**: 逗号分隔的资产类型列表
- **逻辑**:
  ```
  asset_types = []
  IF Q1包含"境外银行存款": asset_types.append("境外银行存款")
  IF Q1包含"境外证券投资": asset_types.append("境外证券投资")
  IF Q1包含"离岸公司": asset_types.append("离岸公司")
  IF Q1包含"境外保单": asset_types.append("境外保单")
  IF Q1包含"离岸信托": asset_types.append("离岸信托")
  
  输出: "、".join(asset_types)
  ```
- **示例**: `境外银行存款、离岸公司、境外保单`

---

### 第二部分：资产总览

#### {xxx_involved}（是否涉及字段）
- **可选值**: `是` | `否`
- **逻辑**:
  ```
  IF 用户在Q1中选择了该资产类型:
    输出: "是"
  ELSE:
    输出: "否"
  ```

#### {xxx_risk_level}（风险等级字段）
- **可选值**: `🔴 高风险` | `🟡 中风险` | `🟢 低风险` | `-`
- **逻辑**:
  ```
  IF 用户未选择该资产类型:
    输出: "-"
  ELSE:
    根据risk_assessment_rules.md中的规则判定风险等级
    输出对应的风险等级图标和文字
  ```
- **示例**:
  - 高风险: `🔴 高风险`
  - 中风险: `🟡 中风险`
  - 低风险: `🟢 低风险`
  - 未涉及: `-`

---

### 第三部分：各资产信息披露义务判定

#### {asset_sections}

这是报告的核心部分，需要为每个涉及的资产类型生成详细的分析章节。

**生成规则**:
```
asset_sections = ""

IF 用户选择了"境外银行存款":
  asset_sections += generate_bank_section()

IF 用户选择了"境外证券投资":
  asset_sections += generate_securities_section()

IF 用户选择了"离岸公司":
  asset_sections += generate_company_section()

IF 用户选择了"境外保单":
  asset_sections += generate_insurance_section()

IF 用户选择了"离岸信托":
  asset_sections += generate_trust_section()

IF 所有资产类型都未选择:
  asset_sections = "*本次诊断未涉及任何资产类型*"
```

**{bank_disclosure_result} 判定逻辑**:

```
根据risk_assessment_rules.md的判定结果:

IF 风险等级 = 高风险:
  输出: "✅ 有披露义务"

ELSE IF 风险等级 = 中风险:
  输出: "⚠️ 可能有披露义务"

ELSE:
  输出: "❌ 无披露义务"
```

**{bank_disclosure_reasoning} 生成逻辑**:

```
生成numbered list，说明判定依据：

高风险示例:
1. 您是中国税收居民
2. 账户开立在{bank_country}，该地区是CRS参与国
3. 账户余额超过100万美元/账户由您本人独有
4. 账户不属于豁免类型
5. **综合判定**：该账户信息将通过CRS交换至中国税务机关

中风险示例:
1. 您是中国税收居民
2. 账户开立在{bank_country}，该地区是CRS参与国
3. 账户余额低于100万美元
4. **综合判定**：虽然余额较低，但仍在CRS申报范围内，信息可能被交换

低风险示例:
1. 账户开立在{bank_country}，该地区不是CRS参与国
2. **综合判定**：无CRS信息交换风险
```

**{bank_exchange_expectation} 生成逻辑**:

```
IF 有披露义务或可能有披露义务:
  输出:
  "
  开户银行将在每年9月30日前：
  1. 识别您为中国税收居民
  2. 向{bank_country}当地税务机关报送以下信息：
     - 您的姓名、地址、税收居民国（中国）、税务识别号（如有）
     - 账户号码
     - 账户余额或价值（截至12月31日）
     - 账户相关的利息、股息等收入（如有）
  3. {bank_country}税务机关将在次年9月前，通过CRS自动交换机制将上述信息传递给中国税务机关
  
  **时间线示例**（以2025年为例）：
  - 2025年12月31日：银行记录账户信息
  - 2026年9月30日前：银行向{bank_country}税务机关报送
  - 2027年9月前：{bank_country}将信息交换给中国税务机关
  "
  
ELSE:
  输出: "暂无CRS信息交换预期。"
```

**{bank_compliance_advice} 生成逻辑**:

```
根据风险等级生成建议：

IF 高风险:
  输出:
  "
  🔴 **高风险情况，建议立即行动**：
  
  1. **主动申报**（虽然目前无强制要求，但建议主动申报以展示合规意愿）
     - 保留账户月结单和年度对账单
     - 在个人所得税年度汇算清缴时，考虑在"境外所得"栏目说明境外账户情况
  
  2. **利息收入纳税**
     - 境外账户的利息收入属于个人所得，需要在中国申报纳税
     - 税率：利息所得适用20%的税率
     - 可抵扣：已在{bank_country}缴纳的利息税可申请抵免
  
  3. **准备应对税务检查**
     - 保留开户文件、资金来源证明
     - 如有大额转入转出，保留相关合同和凭证
     - 建议咨询专业税务顾问，评估潜在的税务影响
  
  4. **关注政策变化**
     - 中国可能未来出台境外账户强制申报规定
     - 建议定期（每年）更新CRS风险评估
  "

ELSE IF 中风险:
  输出中风险建议...（类似结构，措施相对宽松）

ELSE:
  输出低风险建议...
```

**{insurance_tax_treatment} 生成逻辑**:

```
"
### 境外保单的税务处理

**保单现金价值的税务影响**：
1. **保费缴纳阶段**：
   - 保费支出本身不可税前扣除
   - 大额保费支出需关注资金来源的合规性

2. **保单持有期间**：
   - 保单现金价值的增长，在未提取前暂不征税
   - 但保单账户信息会通过CRS交换，税务机关可追踪资产规模

3. **保单分红或领取**：
   - 年金领取、分红提取视为所得，需在中国申报纳税
   - 税率：一般按20%征收个人所得税
```

#### {glossary}
- **格式**: 定义列表
- **内容**: 报告中提到的专业术语解释

**示例**:
```
- **CRS（Common Reporting Standard）**：共同申报准则，由OECD制定的全球税务信息自动交换标准

- **税收居民**：根据一国税法，需要在该国就全球所得纳税的个人或实体。中国税收居民的判定标准主要是在中国境内有住所，或无住所但在一个纳税年度内在中国境内居住累计满183天

- **NFE（Non-Financial Entity）**：非金融机构实体，不属于金融机构的公司、合伙企业等实体

- **Active NFE（积极非金融机构）**：主要从事实质性经营活动的非金融机构，收入或资产50%以上来自主动经营业务

- **Passive NFE（消极非金融机构）**：主要持有金融资产或进行被动投资的非金融机构，需要穿透识别控制人

- **控制人（Controlling Person）**：对实体拥有控制权的自然人，通常指直接或间接持有25%以上股权的个人，或对实体有实际控制影响力的人

- **CFC（Controlled Foreign Corporation）**：受控外国企业，由中国税收居民控制的、设立在低税率地区的外国企业

- **FATCA（Foreign Account Tax Compliance Act）**：美国的《外国账户税收合规法案》，要求全球金融机构向美国IRS报告美国人的账户信息
```
